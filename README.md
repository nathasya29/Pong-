## Description

This project allows you to play pong in the galaxy. It also has a score counter so you can collect the number of points you have with. 
When the ball touches the bottom of the screen the game will quit. See if you can find the secret cheat code..

## How to Run

1. Make sure you have **Python 3.13** installed.
2. Install the dependencies:
   *For most of you, this is just `pip install pygame-ce`.*
3. Run the game:
   ```
   python main.py
   ```
